myFlix Movie Application

myFlix is a web application that provides users with access to information about their favorite films. This includes information about movies, genres, and directors. Users can sign up, update their personal info, and create a list of their favorite movies.

Below are the operations of the myFlix app, the URL endpoints for the API, and correlating response methods.

Request URL HTTP Method Response
Return list of all movies to the user. /movies GET A JSON object holding data about all the movies.
Return data about a single movie by title to the user. /movies/title GET A JSON object holding data about the single movie containing a title and movie data.
Return data about a genre by title. /movies/genre/title GET A JSON object holding data about the movie that was added.
Return data about a director by name. /movies/director GET A JSON string holding data about a director.
Allow new users to register. /users POST A JSON object about the new user registered.
Allow users to update their user info. /users/info PUT A text message indicating user info was updated.
Allow users to add a movie to their list of favorites. /movies/favorites POST A text message indicating a movie was added to their list of favorites.
Allow users to remove a movie from their list of favorites. /movies DELETE A text message indicating a movie was removed from their list of favorites.
Allow existing users to deregister. /users DELETE A text message indicating to the user that they were deregistered.
